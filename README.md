# Bank

## Description
This is a simple project designed to get used to working with Java, specifically interfaces and how to use them to isolate testing. This is also the first time I used a Stream in Java, after looking for a better solution to filter out null items in a List. I still need to do more practice with them but it was good to find out they exist and be able to implement one in my code!

## How to use
``` bash
# Requirements: Java 21
java -version

# Clone the repo
git clone https://github.com/evg4/Bank.git
cd Bank

# Build the project
./gradlew build

# Run the app
./gradlew run

# Run the tests
./gradlew test

```
